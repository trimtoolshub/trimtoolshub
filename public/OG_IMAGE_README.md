# OG Image Requirements

## File: `og-default.png`

**Location:** `/public/og-default.png`

**Specifications:**
- Dimensions: 1200Ã—630 pixels
- File size: < 200KB
- Format: PNG (recommended) or JPG
- Content: Should include TrimToolsHub branding and a visual representation of the tools

**Design Guidelines:**
- Use the brand colors (slate-950 background, brand-accent for highlights)
- Include the TrimToolsHub logo/name
- Show a visual representation of the tools (PDF, QR codes, images, CAD, etc.)
- Keep text minimal and readable
- Ensure it looks good when shared on social media platforms

**Usage:**
This image is used as the default Open Graph image for all pages. Individual pages can override it by passing a custom `ogImage` prop to the `SEOHead` component.

**Note:** Create this image using a design tool (Figma, Canva, etc.) and place it in the `/public` directory.

## Submission Checklist

- [ ] Research each directory's submission guidelines
- [ ] Prepare unique descriptions for each category
- [ ] Gather screenshots and logos
- [ ] Prepare social media links
- [ ] Create submission accounts where needed

- [ ] Track submissions in a spreadsheet
- [ ] Monitor backlinks using Google Search Console
